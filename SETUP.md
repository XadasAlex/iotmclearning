# Quick Setup Guide

## Step 1: Add Your OpenAI API Key

Edit the `.env` file in the project root:

```bash
VITE_OPENAI_API_KEY=sk-your-actual-api-key-here
```

Replace `sk-your-actual-api-key-here` with your real OpenAI API key.

## Step 2: Start the Development Server

```bash
npm run dev
```

## Step 3: Open the Application

Open your browser and go to:
```
http://localhost:5173
```

## That's It!

You should now see the IoT Quiz welcome screen.

## First Time Using?

1. Click "Start Learning Session"
2. Answer the quiz questions
3. Review AI explanations
4. Use the chat feature to ask questions
5. Track your progress as you learn
